# Knuth-Morris-Pratt

An Algorithm For searching a string pattern in a different string

### Input Format

Input: First String
		Second String

### Output Format

Output: (Number Of Occurences)

### Sample Input

```
abcde
abcdeaaaaabcdehlkj
```

### Sample Output

```
2
```
