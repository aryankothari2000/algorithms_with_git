# Matrix Exponentiation

An Algorithm For finding Exponentiation of a Matrix

### Input Format

Input: 
Enter N of (NxN matrix) and Power to be calculated
Enter the matrix elements

### Output Format

Calculated Matrix

### Sample Input

```
2 2
1 1
1 0
```

### Sample Output

```
2 1 
1 1 
```
